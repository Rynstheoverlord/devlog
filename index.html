My top 5 languages

updated 4th March 2023

5. JavaScript: JavaScript only got into the list recently because before now i had utterly loathed and disliked the language, but as time progressed, i realized that JavaScript contained a little beauty within it.

4. Golang: Golang, usually refered to as Go is an outstanding language developed by Google, Athough i havent really written much in golang, i've begun to develop a love for it placing it in my top 4

3. C++: If you're looking for a language that can pretty much do anything and still hold and record for high performance, then C++ is your language. Although this language can be pretty difficult for newbie developers, it is a very good choice for general development.

2. Kotlin: This is another mind blowing language developed by google in an attempt to replace Java in Android development, so if you are trying to become an Android developer, then kotlin is for you, one of my all time favourites finding its way into my top 2
